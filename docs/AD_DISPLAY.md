# サイトへの広告表示ガイド（アドインベントリー）

競馬オッズ計算ツールに**他社の競馬系広告を表示**し、広告収益を得るための設定ガイドです。

## 概要

- **Google AdSense** を利用してサイトに広告を表示
- サイト内容（競馬・馬券計算）に応じて、Google が**競馬関連の広告**を自動配信する可能性あり
- AdSense 審査通過後、出稿待ちの競馬系広告が表示される可能性があります

## 前提

- AdSense は**ギャンブル・賭博に近いコンテンツ**で審査が厳しい場合があります
- 当サイトは「計算ツール」「馬券購入を推奨しない」と明記しているため、審査通過の可能性はあります
- 審査結果は Google の判断によるため、必ずしも承認されるとは限りません

---

## 1. 設定手順

### Step 1: Google AdSense に申請

1. [Google AdSense](https://www.google.com/adsense/) にアクセス
2. Google アカウントでサインイン
3. 「サイトを追加」で `https://keiba-calculator.vercel.app` を登録
4. 審査提出（数日〜数週間）
5. 承認後、広告ユニットを作成

### Step 2: 広告ユニットの作成

1. AdSense ダッシュボード → 広告 → 広告ユニット
2. 「表示広告」を選択
3. 広告サイズ: **レスポンシブ** または **横長（728x90 等）** を推奨
4. 作成後に **広告ユニット ID**（数値）と **発行者 ID**（ca-pub-xxxxxxxxxx）を確認

### Step 3: Vercel に環境変数を設定

| 変数名 | 説明 | 例 |
|--------|------|-----|
| `NEXT_PUBLIC_ADSENSE_CLIENT_ID` | 発行者 ID | `ca-pub-1234567890123456` |
| `NEXT_PUBLIC_ADSENSE_SLOT_ID` | 広告ユニット ID | `1234567890` |

Vercel ダッシュボード → プロジェクト → Settings → Environment Variables で設定し、再デプロイしてください。

---

## 2. 広告の表示箇所

| 場所 | 説明 |
|------|------|
| **フッター上** | 全ページ共通。横長（horizontal）形式 |
| **メインページ中央** | 計算結果と購入リストの間。レスポンシブ（auto）形式 |

追加の広告ユニットが必要な場合は、`AdSlot` に `slotId` を指定して複数配置できます。

---

## 3. 競馬系広告が表示される仕組み

- AdSense は**ページのコンテンツ**を解析し、関連する広告を配信
- 当サイトは競馬・馬券・オッズ・JRA 等のキーワードを含むため、**競馬関連の広告主**が配信対象になる可能性がある
- 広告の種類と表示頻度は Google のアルゴリズムに依存し、保証はありません

---

## 4. 複数広告ユニットを使う場合

異なる広告ユニットを配置したい場合:

```tsx
// 例: メインページとサイド用で別スロット
<AdSlot slotId="1111111111" format="auto" />
<AdSlot slotId="2222222222" format="vertical" />
```

環境変数 `NEXT_PUBLIC_ADSENSE_SLOT_ID_2` などを追加し、`slotId` に渡すこともできます。

---

## 5. 審査が通らない場合

- **AdSense 以外のネットワーク**を検討（例: A8.net などのアフィリエイト、競馬専門メディア向けのアドネットワーク）
- サイトの「計算ツール」としての位置づけをさらに強調（免責の強化、ガイドコンテンツの充実）
