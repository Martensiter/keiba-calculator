## 2. 環境変数設定（Vercel）

Vercel のダッシュボードで以下の環境変数を設定してください。

| 変数名 | 説明 | 例 |
|--------|------|-----|
| `NEXT_PUBLIC_BASE_URL` | 本番URL | `https://keiba-calculator.vercel.app` |
| `NEXT_PUBLIC_GA_MEASUREMENT_ID` | Google Analytics 4 | `G-XXXXXXXXXX` |
| `NEXT_PUBLIC_GOOGLE_ADS_ID` | Google 広告 コンバージョン | `AW-XXXXXXXXXX` |
| `NEXT_PUBLIC_META_PIXEL_ID` | Meta Pixel（Facebook/Instagram） | `XXXXXXXXXXXXXXX` |

※ いずれも省略可能。設定したもののみ読み込まれます。

## 5. 入稿手順（Google 広告）

1. [Google 広告](https://ads.google.com/) にログイン
2. 新規キャンペーン作成
3. 目標: 「ウェブサイトへのアクセス」または「コンバージョン」
4. キャンペーンタイプ: 検索 / ディスプレイ / パフォーマンスmax から選択
5. ランディングページURL: `https://keiba-calculator.vercel.app/`
6. 上記「3. Google 広告 入稿用文言」から見出し・説明文をコピペ
7. コンバージョン設定: GA4 と連携するか、Google 広告タグ（AW-xxx）を設置
8. 審査提出

---

## 6. 入稿手順（Meta 広告）

1. [Meta ビジネススイート](https://business.facebook.com/) にログイン
2. 広告マネージャー → キャンペーン作成
3. 目標: 「トラフィック」「エンゲージメント」など
4. ランディングページURL: `https://keiba-calculator.vercel.app/`
5. 上記「4. Meta 広告 入稿用文言」からテキスト・見出しをコピペ
6. Meta Pixel を設置（環境変数 `NEXT_PUBLIC_META_PIXEL_ID` を設定済みなら自動）
7. 審査提出

---

## 7. 注意事項

- **ギャンブル・競馬**: プラットフォームによって広告制限があります。当サイトは「計算ツール」であり馬券購入を推奨していない旨を明確にしているため、審査通過の可能性はありますが、拒否された場合は各ポリシーを確認してください。
- **18歳未満**: 広告テキストや配信ターゲットで、18歳未満を対象としないよう注意してください。
- **プライバシーポリシー**: 広告審査では必須です。`/privacy` のリンクをフッターに設置済みです。

## 8. コンバージョン計測（オプション）

特定のアクション（例: 購入リストへの追加）をコンバージョンとして計測したい場合は、`AnalyticsScripts` を参考にカスタムイベントを送信できます。

```typescript
// GA4 カスタムイベント例
if (typeof window !== 'undefined' && window.gtag) {
  window.gtag('event', 'add_to_list', { value: 1 });
}
```
