# 🚀 5分で始める競馬オッズ計算ツール開発

このガイドに従えば、**モバイル端末だけで**今すぐ開発を始められます！

## 📱 必要なもの

- スマートフォンまたはタブレット
- GitHubアカウント
- インターネット接続

**それだけ！PCは不要です。**

---

## ステップ1：Codespacesを起動（1分）

### スマホ・タブレットで操作

1. **GitHubリポジトリにアクセス**
   
   ブラウザ（Chrome、Safari、Firefoxなど）で以下にアクセス：
   ```
   https://github.com/Martensiter/keiba-calculator
   ```

2. **Codeボタンをタップ**
   
   画面上部の緑色の「**Code**」ボタンをタップします。

3. **Codespacesタブを選択**
   
   表示されたメニューで「**Codespaces**」タブをタップ。

4. **Create codespaceをタップ**
   
   「**Create codespace on [ブランチ名]**」ボタンをタップ。
   
   ⏳ 30秒〜2分ほど待ちます（初回のみ）

5. **完了！**
   
   VS Codeがブラウザで起動します 🎉

---

## ステップ2：開発環境を準備（2分）

Codespacesが起動したら、画面下部の**ターミナル**が表示されています。

### 自動インストール

初回起動時、依存関係が自動的にインストールされます。

```
🎯 競馬オッズ計算ツール開発環境へようこそ！

開発サーバーを起動するには: npm run dev
ビルドするには: npm run build
```

このメッセージが表示されたら準備完了です！

---

## ステップ3：開発サーバーを起動（1分）

ターミナルに以下を入力：

```bash
npm run dev
```

するとこのように表示されます：

```
▲ Next.js 14.2.35
- Local:    http://localhost:3000
- Network:  http://0.0.0.0:3000

✓ Ready in 1234ms
```

### プレビューを開く

画面下部の「**PORTS**」タブをタップ：

1. ポート「**3000**」の行を探す
2. 「🌐」アイコンをタップ
3. 「**Open in Browser**」を選択

新しいタブでアプリが開きます！🚀

---

## ステップ4：コードを編集してみる（1分）

### ファイルを開く

1. 左サイドバーの「**EXPLORER**」アイコンをタップ
2. `src` → `app` → `page.tsx` を開く

### 変更してみる

たとえば、36行目あたりの：

```typescript
const [betType, setBetType] = useState<BetType>('馬連');
```

を以下に変更：

```typescript
const [betType, setBetType] = useState<BetType>('単勝');
```

**保存すると自動的にアプリがリロードされます！**

プレビューを確認すると、初期選択が「単勝」に変わっています。

---

## 🎉 完了！

これであなたもモバイル開発者です！

### 次にやること

#### コードを元に戻す
先ほどの変更を元に戻してみましょう。

#### Git操作を試す
1. 左サイドバーの「**Source Control**」アイコン（分岐のマーク）をタップ
2. 変更したファイルが表示されます
3. メッセージ欄に「テスト変更」と入力
4. 「**✓ Commit**」ボタンをタップ
5. 「**...**」メニューから「**Push**」を選択

#### 他のファイルを探索
- `src/components/calculator/` - UIコンポーネント
- `src/lib/calculator/` - 計算ロジック
- `src/app/globals.css` - スタイル定義

## 💡 モバイル開発のコツ

### 画面を横向きに
タブレットの場合、横向きにするとコードとプレビューを並べて表示できます。

### キーボードショートカット
- ファイル検索: 画面上部の検索バーをタップ
- コマンドパレット: ⋮ メニュー → Command Palette
- 保存: 自動保存がONです（変更不要）

### 外付けキーボード
Bluetoothキーボードを接続すると、PC並みの快適さになります！

## 🆘 困ったときは

### プレビューが表示されない
```bash
# ターミナルで再起動
# Ctrl+C を押してから
npm run dev
```

### Codespacesが重い
- 他のタブやアプリを閉じる
- ブラウザを再起動
- Codespacesを停止して再起動

### わからないことがある
- `MOBILE_DEV.md` - 詳細ガイド
- `README.md` - プロジェクト概要
- `CLAUDE.md` - 技術仕様
