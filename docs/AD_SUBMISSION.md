# 競馬系広告入稿ガイド

競馬オッズ計算ツールの Google 広告・Meta 広告・Yahoo! プロモーション広告 入稿用ドキュメントです。
競馬・馬券関連はプラットフォームごとに制限・認証が必要なため、ポリシー調査結果と対応策をまとめています。

## 1. 各プラットフォームの競馬・ギャンブル広告ポリシー

### Google 広告（ギャンブル、ゲーム）

- **ポリシー**: [ギャンブル、ゲーム - Google 広告ポリシー](https://support.google.com/adspolicy/answer/6015406?hl=ja)
- **概要**:
  - ギャンブル・賭博関連の広告は**国・地域ごとに制限**が異なる
  - **認証が必要**な地域では、Google による事前認証を受けた広告主のみが配信可能
  - 日本は認証対象地域に含まれる可能性が高く、競馬（JRA等）もギャンブルカテゴリに該当しうる
- **当サイトの位置づけ**:
  - 「馬券の購入を推奨しない」「計算ツール」「シミュレーション」として、**賭博促進ではなく情報提供・ツール**と主張可能
  - 審査時に「計算機」「オッズ確認ツール」として説明すると通過しやすい場合がある
- **推奨対応**:
  1. 最初は**控えめな文言**（計算ツール・シミュレーション特化）で申請
  2. 拒否された場合は [ポリシー免除申請](https://support.google.com/google-ads/answer/9841640) または認証プロセスを確認
  3. 広告文言に「馬券購入は推奨しません」「20歳以上」を明記

---

### Meta 広告（Facebook / Instagram）

- **ポリシー**: [ギャンブルとゲーム - Meta 広告ポリシー](https://www.facebook.com/policies/ads/restricted_content/gambling_and_games)
- **概要**:
  - ギャンブル・ゲーム関連は**特別広告カテゴリ（Special Ad Category）**に該当
  - **事前の書面による許可**が必要な地域がある
  - 年齢制限（18歳以上など）のターゲティングが必須
  - 日本向けの詳細は [Meta ビジネスヘルプ](https://www.facebook.com/business/help) で確認
- **当サイトの位置づけ**:
  - 馬券販売サイトではなく「計算ツール」であることを強調
  - ギャンブルカテゴリとして扱われる場合は、特別広告カテゴリを選択し申請
- **推奨対応**:
  1. キャンペーン作成時に「ギャンブルとゲーム」の特別広告カテゴリを選択（該当する場合）
  2. ターゲティングを**18歳以上**に限定
  3. 広告文言に責任ある利用を促す表現を入れる

---

### Yahoo! プロモーション広告（Yahoo! JAPAN）

- **概要**:
  - 国内向けサービスであり、競馬（JRA）は公営競技として社会的に認知されている
  - Google・Metaより**競馬系広告の許容度が高い**可能性がある
  - 詳細は [Yahoo! プロモーション広告](https://promotionalads.yahoo.co.jp/) の利用規約・ガイドラインを確認
- **推奨**:
  - 競馬ファンへのリーチを重視する場合、**Yahoo! を優先検討**
  - 検索連動型で「競馬 オッズ 計算」などのキーワードに強い

## 2.  positioning 戦略：計算ツールとしての打ち出し

審査通過率を高めるため、以下の positioning を推奨します。

1. **「馬券の購入を推奨しない」を明記** … サイト・広告の両方で表示済み
2. **「計算ツール」「シミュレーション」を前面に** … 賭博促進ではなく情報提供
3. **「20歳以上」「自己責任」を記載** … 責任ある利用の啓発
4. **JRA・即PAT は「データ入力の便宜」として言及** … 投票促進ではなく入力効率化

## 6. 入稿手順

### Google 広告

1. [Google 広告](https://ads.google.com/) にログイン
2. 新規キャンペーン作成
3. 目標: 「ウェブサイトへのアクセス」
4. ランディングページ: `https://keiba-calculator.vercel.app/`
5. 文言: 審査が厳しい場合は **A**、余裕があれば **B** から選択
6. ターゲット: **18歳以上**を推奨
7. 審査提出。拒否時は [再審査請求](https://support.google.com/google-ads/answer/9841640) を検討

### Meta 広告

1. [Meta ビジネススイート](https://business.facebook.com/) にログイン
2. 広告マネージャー → キャンペーン作成
3. 「ギャンブルとゲーム」の特別広告カテゴリが必要か確認
4. ランディングページ: `https://keiba-calculator.vercel.app/`
5. ターゲット: **18歳以上**に設定
6. 文言: **A** または **B** から選択

### Yahoo! プロモーション広告

1. [Yahoo! プロモーション広告](https://promotionalads.yahoo.co.jp/) に登録
2. キャンペーン作成
3. キーワード: 「競馬 オッズ 計算」「馬券 点数 計算」「3連単 計算」など
4. 文言: **B** を推奨
